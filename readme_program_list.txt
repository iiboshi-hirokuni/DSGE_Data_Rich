program list

ceemh_data_rich.g:  main program of MH algorithm of data rich DSGE model of Christiano, Eichenbaum and Evans (CEE)(2005) 

cee_data_rich_smoother.src: simulation smoother and other routines of MCMC.

cee_lh_dr.src: calculating likelihood function of Data Rich DSGE

hist_decomp_save.src:  saving a file of historical decompostion.

ceepm.g: main program of calculating posterior mode of regular DSGE of CEE type 

ceehess.g: The program computes the hessian at the posterior mode 

ceemod.src: computing rational expectations of CEE (2005) type DSGE model using gensys.
